## はじめに
このリポジトリは配布マップ[TheSkyBlessing](https://github.com/ProjectTSB/TheSkyBlessing)の非公式拡張データパックの製作・配布場所です。  
本データパックを利用中にバグが発生した場合、最下部記載の各種リンクからバグの報告をお願いします。
